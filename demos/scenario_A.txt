# 데모 시나리오 A: 신규 판매자 가입 및 상품 등록

## 시나리오 개요
신규 판매자가 스마트스토어 가입부터 첫 상품 등록까지의 과정을 문의하는 시나리오

## 대화 흐름

### 1단계: 가입 조건 문의
**사용자**: "미성년자도 판매 회원 등록이 가능한가요?"

**챗봇**: "안녕하세요! 스마트스토어 전문 상담사입니다.

네이버 스마트스토어는 만 14세 미만의 개인(개인 사업자 포함) 또는 법인사업자는 입점이 불가함을 양해 부탁 드립니다.

**가입 가능 조건:**
- 만 14세 이상 개인
- 개인사업자 또는 법인사업자
- 사업자등록증 소지자

**필요 서류:**
1. 사업자등록증 (개인사업자 또는 법인)
2. 신분증 (주민등록증, 운전면허증 등)
3. 통장사본

더 궁금한 점이 있으시면 언제든 물어보세요!

<followups>
- 등록에 필요한 서류 안내해드릴까요?
- 등록 절차는 얼마나 오래 걸리는지 안내가 필요하신가요?
- 개인사업자 등록 방법이 궁금하신가요?
</followups>"

### 2단계: 가입 절차 문의
**사용자**: "등록 절차는 얼마나 오래 걸리는지 안내가 필요하신가요?"

**챗봇**: "스마트스토어 가입 절차와 소요 시간을 안내해드릴게요!

**가입 절차 및 소요 시간:**

1. **서류 준비** (1-2일)
   - 사업자등록증 발급 (개인사업자의 경우)
   - 신분증, 통장사본 준비

2. **온라인 신청** (10-15분)
   - 스마트스토어 가입 페이지 접속
   - 기본 정보 입력 및 서류 업로드

3. **심사 과정** (1-3일)
   - 서류 검토 및 승인
   - 계정 활성화

4. **첫 상품 등록** (30분-1시간)
   - 상품 정보 입력
   - 이미지 업로드
   - 가격 및 배송 정보 설정

**총 소요 시간: 약 3-7일**

**팁:** 서류가 완벽하게 준비되어 있으면 심사 시간이 단축될 수 있습니다.

<followups>
- 상품 등록 방법을 자세히 알고 싶으신가요?
- 첫 상품 등록 시 주의사항이 궁금하신가요?
- 상품 승인 과정에 대해 궁금하신가요?
</followups>"

### 3단계: 상품 등록 문의
**사용자**: "상품 등록 방법을 자세히 알고 싶으신가요?"

**챗봇**: "상품 등록 방법을 단계별로 안내해드릴게요!

**상품 등록 절차:**

1. **스마트스토어 센터 접속**
   - 로그인 후 '상품' → '상품 등록' 선택

2. **기본 정보 입력**
   - 상품명 (검색에 중요한 키워드 포함)
   - 카테고리 선택 (정확한 카테고리 필수)
   - 브랜드명

3. **상품 이미지 업로드**
   - 메인 이미지: 800x800px 이상 권장
   - 상세 이미지: 최대 20장까지
   - 이미지 품질이 판매에 직접 영향

4. **상품 상세 정보**
   - 상품 설명 (HTML 에디터 사용 가능)
   - 옵션 설정 (색상, 사이즈 등)
   - 재고 수량

5. **가격 및 배송 정보**
   - 판매가, 할인가 설정
   - 배송 방법 및 비용
   - 배송 지역 설정

6. **등록 완료**
   - 미리보기로 확인 후 등록
   - 승인 후 판매 시작

**주의사항:**
- 상품명에 브랜드명 포함 금지
- 실제 상품과 다른 이미지 사용 금지
- 가격 정책 준수 필수

<followups>
- 상품 이미지 업로드 규격이 궁금하신가요?
- 카테고리 선택 기준을 알고 싶으신가요?
- 상품 승인 과정에 대해 궁금하신가요?
</followups>"

## 시나리오 특징
- 자연스러운 대화 흐름
- 단계별 상세 안내
- 관련 후속 질문 제안
- 실용적인 정보 제공
