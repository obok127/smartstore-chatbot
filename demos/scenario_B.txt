# 데모 시나리오 B: 정산 및 수수료 문의

## 시나리오 개요
기존 판매자가 정산 주기, 수수료, 세금계산서 등 정산 관련 업무를 문의하는 시나리오

## 대화 흐름

### 1단계: 정산 주기 문의
**사용자**: "정산 주기와 수수료는 어떻게 되나요?"

**챗봇**: "안녕하세요! 스마트스토어 정산 관련 정보를 안내해드릴게요.

**정산 주기:**
- **일반 정산**: 매월 15일, 말일 기준으로 2회 정산
- **빠른 정산**: 판매 후 3-5일 내 정산 가능 (수수료 추가)
- **정산 지연**: 주말/공휴일은 다음 영업일로 연기

**수수료 구조:**
- **기본 수수료**: 카테고리별 2-15% (의류 5%, 전자제품 3% 등)
- **결제 수수료**: 별도 2.5% (네이버페이)
- **빠른 정산 수수료**: 추가 1-2%

**정산 금액 계산:**
판매가 - (판매가 × 수수료율) - 배송비 = 정산 금액

**예시:** 10만원 상품 판매 시
- 수수료 5%: 5,000원
- 결제 수수료 2.5%: 2,500원
- 정산 금액: 92,500원

<followups>
- 특정 카테고리 수수료율이 궁금하신가요?
- 세금계산서 발급 방법을 알고 싶으신가요?
- 정산 지연 사유를 확인하고 싶으신가요?
</followups>"

### 2단계: 세금계산서 문의
**사용자**: "세금계산서 발급 방법을 알고 싶으신가요?"

**챗봇**: "세금계산서 발급 방법을 안내해드릴게요!

**세금계산서 발급 절차:**

1. **스마트스토어 센터 접속**
   - 로그인 후 '정산관리' → '세금계산서 조회' 선택

2. **발급 기간 설정**
   - 월별 또는 분기별 선택
   - 발급하고자 하는 기간 설정

3. **세금계산서 생성**
   - '세금계산서 발급' 버튼 클릭
   - 시스템에서 자동 생성

4. **다운로드 및 보관**
   - PDF 형태로 다운로드
   - 사업자등록번호별로 분리 발급

**주의사항:**
- **발급 시점**: 정산 완료 후 1-2일 후 발급 가능
- **보관 의무**: 5년간 보관 필수
- **수정 불가**: 발급 후 수정 불가능

**자주 묻는 질문:**
- Q: 분기별로 발급 가능한가요?
- A: 네, 월별/분기별/연간 선택 가능합니다.

- Q: 세금계산서 수정이 필요한 경우는?
- A: 발급 후 수정 불가하므로 정산 내역 확인 후 발급하세요.

<followups>
- 부가세신고 내역 조회 방법이 궁금하신가요?
- 정산 내역 상세 확인 방법을 알고 싶으신가요?
- 수수료 계산 방법을 자세히 알고 싶으신가요?
</followups>"

### 3단계: 정산 지연 문의
**사용자**: "정산이 지연되고 있는데 확인해보고 싶어요"

**챗봇**: "정산 지연 확인 방법을 안내해드릴게요!

**정산 지연 확인 절차:**

1. **정산 상태 확인**
   - '정산관리' → '정산 내역' 접속
   - 해당 월의 정산 상태 확인

2. **지연 사유 확인**
   - **서류 미제출**: 사업자등록증, 통장사본 등
   - **정보 불일치**: 계좌번호, 예금주명 오류
   - **정산 보류**: 환불/교환 건 처리 중
   - **시스템 점검**: 정산일이 공휴일인 경우

3. **해결 방법**
   - **서류 제출**: 누락된 서류 즉시 제출
   - **정보 수정**: 정산 계좌 정보 정확히 입력
   - **고객센터 문의**: 1588-3819

**정산 지연 예방 팁:**
- 매월 10일까지 서류 상태 확인
- 정산 계좌 정보 정확성 체크
- 환불/교환 건 사전 처리

**정산 일정:**
- 1월 15일: 12월 16일~31일 판매분
- 1월 31일: 1월 1일~15일 판매분
- 2월 15일: 1월 16일~31일 판매분

<followups>
- 정산 계좌 정보 변경 방법이 궁금하신가요?
- 환불/교환 처리 방법을 알고 싶으신가요?
- 고객센터 연락처를 확인하고 싶으신가요?
</followups>"

## 시나리오 특징
- 실무적인 정산 업무 안내
- 구체적인 수치와 예시 제공
- 문제 해결 방법 제시
- 예방적 조치 안내
